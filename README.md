## About the Project

Instead of copy past the function in my script manually I make a collection which I can directly add to my `bashly` 
projects.

## Getting Started

### Prerequisites

- Bashly see bashly.dev

### Installation

- Just clone it via `$ bashly add github-link`

## Usage

Each functions has it own file. You can call those functions in your main bash script files.
